### 3. 기능

피싱체크는 다음과 같은 기능을 제공합니다:

- 문자 분석을 통해 스미싱 여부 확인
- 문자화면 캡쳐 분석을 통해 스미싱 여부 확인
- 스미싱 수법을 알려주는 선택형 챗봇
- 보이스피싱/스미싱 대응 방법
- 새로운 종류의 스미싱 수법 관련하여 금융감독원의 '소비자경보' 공지사항 조회
- 각 금융기관의 전화번호 안내 및 홈페이지 연결
- 개발자에게 보내는 패드백
- 모바일 앱 설치(PWA)
